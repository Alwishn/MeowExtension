1. Загрузите архив Project.rar
2. Откройте браузер и введите в адресную строку browser://extensions/
3. Включите режим Разработчика в правом верхнем углу технической вкладки
4. Распакуйте в любую директорию Архив Project.rar
5. В браузере, нажмите кнопку "Загрузить распакованное расширение" и выберите папку Project.
6. Разрешите расширению взаимодействие с сайтами.
